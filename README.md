# DASTNET Traffic Forecasting

A traffic speed forecasting dashboard built using DASTNET model, 
evaluated on METR-LA and PEMS-BAY benchmarks.

## Demo
Run the Streamlit dashboard via the notebook on Kaggle.

## Models
- DASTNET (main model)
- ASTGCN, DCRNN, STGCN (baselines)

## Datasets
- [METR-LA](https://kaggle.com/datasets/YOUR_USERNAME/traffic-weights)
- [PEMS-BAY](https://kaggle.com/datasets/YOUR_USERNAME/traffic-weights)

## Weights
Pretrained weights available at: https://kaggle.com/datasets/YOUR_USERNAME/traffic-weights

## Requirements
- Python 3.6+
- PyTorch
- Streamlit
- pyngrok
